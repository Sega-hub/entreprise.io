Модуль Сканер

  • Сканирование кода по
    следующим показателям:
  • - Наличие дефектов
  • - Контроль уязвимостей
  • - Контроль синтаксиса
  • - Проверка
    повторяемости
  • - Контроль сложности
    поддержки
  • - Анализ размера проекта
  • Поддержка режимов
    сканирования:
  • - Git
  • - SVN
  • - SCP
  • Наборы правил
    сканирования для всех
    языков программирования
  • Конфигурация
    сканирования
  • Управление порогами
    качества
  • Пакетное сканирование
  • Сканирование по
    расписанию

Модуль Аналитика

  • Сбор метрик
    производственных
    процессов в соответствии с
    лучшими практиками
    индустрии
  • Вычисление ключевых
    индикаторов
    производственного
    процесса
  • Настраиваемые
    пользовательские панели
    индикаторов
  • Сквозная визуализация
    потоков создания ценности
  • Декомпозиция данных в
    отчетах до уровня
    первичных метрик
  • Анализ производительности
    команд и отдельных
    сотрудников
  • Стандартизация
    инженерных возможностей

Модуль Задачи

  • Управление задачами
  • Иерархия и взаимосвязи
    между задачами
  • Настройка отображения
    списка задач:
  • - Проcтой список
  • - Иерархическая
       структура
  • - Диаграмма Ганта
  • - Канбан-доска
  • Инструменты фильтрации,
    группировки и сортировки
    списка задач
  • Встроенный язык запросов
    для формирования выборок
    (IQL)
  • Инструменты SCRUM:
  • - Создание спринтов
  • - Планирование спринтов
  • - Отслеживание статусов
        задач
  • - Закрытие спринтов
  • Настройка рабочих
    процессов
  • Автоматизация рабочих
    процессов
  • Экспорт-импорт задач
  • Операционная статистика по
    работе проектов

Модуль Конвейер

  • Конструктор сборочного
    конвейера
  • Шаблоны сборки для языков
    программирования:
  • Настройка отображения
    списка задач:
  • - Java
  • - Node.js
  • - Python
  • - Golang
  • Поддержка режимов
    сканирования:
  • - GIT
  • - SVN
  • - SCP
  • Сценарии сборочных
    конвейеров:
  • - Непрерывная
    интеграция (CI)
  • - Непрерывное
    развертывание (CD)
  • - Контроль качества
    (Quality gates)
  • - Запуск скриптов
  • Библиотека
    пользовательских шаблонов
  • Процесс согласования
    сборки
  • Расписание сборок
  • Статистика запуска
    сборочных конвейеров

Модуль Код

  • Корпоративный репозиторий
    кода
  • Система контроля версий Git
  • Совместная работа с кодом
  • Управление доступом
  • Рецензирование кода (code
    review)
  • Управление запросами на
    слияние
  • Поддержка методов слияния:
  • - Стандартное
       объединение
  • - Сквош
  • - Слияние перемоткой
       (fast-forward)
  • - Перебазирование
       (rebase)
  • - Выборочное слияние
       (cherry pick)
  • Защита веток
  • Поддержка вебхук
  • Управление релизам
  • Статистика по коммитам
  • Журнал событий

Модуль Вики

  • Корпоративная база знаний
  • Управление библиотеками
    документов
  • Создание и хранение
    документов следующих
    типов:
  • - Текстовые документы
       в формате markdown
  • - Текстовые документы
       в формате Word
  • - Электронные таблицы
  • - Презентации
  • - Диаграммы
  • Система версионирования
    документов
  • Загрузка документов из
    внешних источников
